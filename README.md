# 기장군 · 부산광역시

> 기장군 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-busan-gijang.github.io/**

기장군은 부산광역시 유일의 군(郡) 단위 기초자치단체로, 동해와 맞닿은 긴 해안선을 따라 어촌과 신도시가 공존하는 독특한 지역입니다. 대변항에서 갓 잡아 올린 멸치와 미역, 다시마가 밥상에 오르는 수산의 고장이자, 고리원자력발전소와 정관·일광 신도시가 들어선 산업·주거 복합 지역이기도 합니다. 해동용궁사와 죽성성당, 오시리아 관광단지처럼 바다를 배경으로 한 명소가 이어져 한 해 수백만 명이 찾는 부산 동부의 관문 역할을 합니다. 1995년 부산광역시로 편입되며 옛 기장현의 이름을 되살린 이곳은 전통 어업과 첨단 산업, 관광이 한자리에서 만나는 보기 드문 고장입니다. 베틀을 차린 형국이라 일컬어지는 지형 위에서 옛 정취와 새로운 도시의 활력이 나란히 자라나고 있습니다.
